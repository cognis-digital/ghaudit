ghaudit checks whether a GitHub organization is configured securely. You give it an export file from your GitHub org, and it tells you what is risky — things like accounts without two-factor authentication, repositories with no branch protections, or secrets that haven't been rotated in over a year. It produces a plain-English report you can read in the terminal, export as JSON for automation, or view as an HTML page. It is built for security teams, developers, and org owners who want a quick, no-setup way to find the most dangerous gaps before attackers do.
